# Admin Panel

Professional React + TypeScript + Tailwind CSS admin panel for managing users, calls, campaigns, referrals, balances, and platform settings.

## Tech Stack

- **Frontend**: React 18, TypeScript 5
- **State Management**: Redux Toolkit
- **Styling**: Tailwind CSS 3
- **Routing**: React Router DOM 6
- **HTTP Client**: Axios
- **Forms**: React Hook Form + Zod
- **Charts**: Recharts
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Testing**: Vitest + React Testing Library + Playwright

## Prerequisites

- Node.js >= 18.0.0
- npm >= 9.0.0
